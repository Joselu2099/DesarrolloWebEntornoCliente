Madrid Digital Project

Un vistazo completo a cómo desarrollé el Proyecto Madrid Digital.

Inicio del Proyecto

Este proyecto fue concebido con el objetivo de digitalizar diversos servicios del gobierno de Madrid, permitiendo a los ciudadanos acceder a información de forma más eficiente.

Objetivos

Nuestro objetivo principal fue mejorar la accesibilidad de los servicios públicos de Madrid, desarrollando una interfaz moderna y fácil de usar para los ciudadanos.

Características Destacadas

A continuación, puedes ver una serie de capturas de pantalla que muestran las características clave que desarrollé para el proyecto.

Interfaz de Usuario

La interfaz permite una navegación fluida entre los distintos servicios de Madrid Digital, con un diseño claro y accesible.

Mapa Interactivo

Implementé un mapa interactivo que facilita la búsqueda de recursos e información pública dentro de la ciudad.

Accesibilidad

Cada página fue diseñada con el objetivo de ser completamente accesible, incluso para personas con discapacidades visuales.

Tecnologías Utilizadas

  • HTML5, CSS3, JavaScript
  • React para la construcción de la interfaz de usuario
  • Leaflet para el mapa interactivo
  • Node.js y Express para el backend
  • MongoDB para almacenar la información

Conclusiones

El Proyecto Madrid Digital ha sido una experiencia invaluable para mejorar la accesibilidad a los servicios públicos de la ciudad. El uso de tecnologías modernas ha permitido ofrecer una solución robusta y eficiente.
